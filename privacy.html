Privacy Policy for ChatGuard

ChatGuard may collect and store Discord user IDs, server IDs, channel IDs, role IDs, moderation logs, verification logs, warnings, bans, and bot configuration data.

This data is used only to provide bot features such as verification, moderation, logging, and server protection.

ChatGuard does not sell user data to third parties.

Server administrators can request deletion of their server data by contacting the bot owner.

Data may be deleted when the bot is removed from a server or when it is no longer needed.
